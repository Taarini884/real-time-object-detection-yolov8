#  Real-Time Object Detection using YOLOv8

 A real-time object detection web application built using **YOLOv8, Streamlit, and WebRTC**, capable of detecting objects, tracking counts, and triggering alerts dynamically.

---

##  Features

* Real-time webcam object detection
* Live object counting
* Smart alert system for selected objects
* Adjustable confidence threshold
* Video recording functionality

---

##  Tech Stack

* Python
* YOLOv8 (Ultralytics)
* Streamlit
* OpenCV
* Streamlit-WebRTC
* Pandas

---

##  How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
